나 엄진영. 강의 시간이 곧 나의 즐거움. 이해할 때까지 가르친다. 포기란 없다.

어떤 것을 잘 하고 싶다면 먼저 그 것에 대한 깊은 이해가 있어야 합니다.

잘 이해하기 위해서는 먼저 익숙해져야 하고,

어떤 것에 익숙해지는 데는 많은 연습이 필요합니다.

많은 연습을 통해 익숙해지고,

이해도가 높아지다 보면 프로그래밍을 잘하게 될 것입니다.

RECORD

  • 1985

    인생 첫 컴퓨터, 금성(GoldStar, 현재 LG) MSX 컴퓨터 구입

    BASIC 언어로 책에 있는 예제를 몇 번 치다가 그만 둠

    주로 카트리지 꼽아서 게임하는 용도로 사용
    돼지 저금통을 깨서 동네 형한테 게임 카트리지를 사기도 함

  • 1988

    인텔 8088 CPU를 채용한 Trigem XT 컴퓨터 구입
    플로피 디스크(5.25인치)와 하드 디스크(20MB)라는 신세계를 경험
    MS-DOS와 삼보 보석글을 만나다
    동네 학원에서 베이직 배움

  • 그 이후 -> 전자계산학과에서 프로그래밍 토대를 갖춤

    GW Basic, Cobol, Fortran을 배우다
    Borland의 Turbo C 2.0 입문
    Windows 3.1을 만나다
    Board C++ 3.x로 C++ 프로그래밍 입문
    Visual C++ 1.0으로 갈아타다
    학교 실습실 컴에 Windows 95 평가판을 깔다
    윈도 프로그래밍에 뛰어들다
    현대전자 주최 전국 대학생/대학원생 소프트웨어 경진대회에서 동상 입상
    Visual Basic 과의 만남
    Windows NT에서 IIS와 ASP로 웹 프로그래밍을 처음 접하다

  • 그 이후 -> 직업으로 프로그래머가 되다

    첫 직장은 도서관리 시스템 개발 업체
    시청, 구청, 대학교 홈페이지 유지 보수 업체로 옮김
    분산 웹메일 아이템으로 모교 창업보육센터에 입주
    민영 의료보험 중계 시스템을 개발하는 벤처기업에 입사
    중소기업용 그룹웨어 서비스 개발 회사에 다님

  • 그 이후 -> 프로그래밍 강사가 되다

    현대정보기술 교육센터에서 자바 프로그래밍 강의 시작
    쌍용교육센터, 삼성SDS 멀티캠퍼스, SUN 교육센터, 신한은행, 하나은행, 현대중공업, 현대오토에버, 티맥스소프트, 하이닉스 등
    프리랜서 개발자로 나서다
    개발하다 지치면 강사로, 개발이 그리우면 개발자로
